# Importable Examples

These files are small, inactive n8n workflow examples using placeholders only. Import into a test workspace, attach credentials through n8n Credentials, verify node parameters against your n8n version, and test before activation.

- `lead-webhook-to-google-sheets.json`
- `shopify-order-to-slack.json`
- `api-health-monitor.json`
- `central-error-alert.json`
- `ai-email-summary.json`
- `postgres-incremental-sync.json`
